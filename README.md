# 🎮 Sistema de Gestión de Torneo

## 📌 Descripción

Este proyecto implementa un sistema de gestión de torneos de videojuegos en Java. Permite registrar jugadores, simular partidas de forma aleatoria y consultar estadísticas.

## ⚙️ Funcionalidades

* Alta de jugadores
* Listado de jugadores
* Búsqueda por nombre
* Simulación de partidas
* Historial de partidas
* Ranking de jugadores

## 🧱 Estructura del proyecto

* `Torneo`: Administra jugadores y partidas
* `Partida`: Simula enfrentamientos
* `Jugador`: Representa un jugador
* `Estadisticas`: Maneja puntajes y partidas

## 🛠️ Tecnologías utilizadas

* Java
* BlueJ

## ▶️ Ejecución

Ejecutar la clase `Main` para iniciar el sistema.
